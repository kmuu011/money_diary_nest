# 프로젝트 설명
Nest JS 기반의 프로젝트입니다.

# 환경 설정
| 프로그램 | 버전 |
| ------ | ------ |
| Node.js | v14.15.1 |
| npm | v6.14.8 |

# 커밋 타입
| 키워드 | 설명 |
| ------ | ------ |
|feat | 새로운 기능 |
|fix | 버그 수정|
|refactor | 리팩토링|
|css | css 관련 퍼블리싱 수정|
|style | 코드 형식, 정렬, 주석 등 동작에 영향을 안주는 변경사항|
|test | 테스트 코드 관련 내용|
|docs | 문서 수정 (제품 코드 수정 X)|
|etc | 이외 위에서 해당하지 않는 모든 수정사항|
